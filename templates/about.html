  4. initiate DB cd /home/timelapsecontroller/db ; python setup.py
  5. set-up supervisord - check /home/timelapsecontroller/supervisor scripts - to go in /etc/supervisor/conf.d
  6. Should be close ish to getting it working on restart of supervisor
  7. Browse to http://<your.lan.ip>:5000/
