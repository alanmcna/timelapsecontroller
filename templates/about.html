Ready to shoot

Designed to run standalone on a pi running raspbian.

Set-up

  1. git clone ...
  2. (optional) sudo ln -s timelapsecontroller /home/timelapsecontroller
  3. cd timelapsecontroller
  4. python timelapsecontroller.py
  5. Browse to http://127.0.0.1:5000/ # assumes no other flash instances or processes using port 5000
  6. (optional) sudo python bin/focusandshoot.py
  7. ... test ...
  8. (optional) To fully automate imstall supervisord - and review ./supervisor scripts - to go in /etc/supervisor/conf.d
