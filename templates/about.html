Overview

Ready to shoot

Designed to run standalone on a pi running raspbian.

Set-up

  1. get the code (clone the git repo) 
    git clone https://github.com/alanmcna/timelapsecontroller.git
cd timelapsecontroller
  2. Initialise the DB 
    $ python
>>> from timelapsecontroller import init_db
>>> init_db()
  3. Run up the app 
    python timelapsecontroller.py
  4. Browse to http://127.0.0.1:5000/
  5. (optional) Start up the back-end process (if camera connected) 
    sudo python bin/focusandshoot.py

To fully automate imstall supervisord - and review ./supervisor scripts - to go in /etc/supervisor/conf.d
